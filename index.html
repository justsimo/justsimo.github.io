Data Mining project on Expected Goals (xG) for shots from the Top-5 European leagues 2017/2018. 👥 Social Network and Media Analysis

Twitter sentiment analysis for COP26/27 using two methods: VADER (lexicon, rule-based) and a transformer model Twitter-RoBERTa. The pipeline includes dataset harmonization, text cleaning (URL/prefix/suffix removal, emoji normalization), hashtag extraction, user geolocation parsing, and time-series analysis. We compare model outputs (trends, distributions) and build word clouds for semantic insight; Linear Regression and Decision Tree Regressor are used to interpret neg/neu/pos sentiment components.

🤖 Machine Learning & Deep Learning

Research project on using transformers to compute the Greatest Common Divisor (GCD) as a supervised translation task: integer pairs are tokenized in base B and the model outputs the GCD. We analyze how encoding base and training distributions (uniform, log-uniform, etc.) impact accuracy, count of correct GCD ≤100, and grokking, evaluating on both natural and stratified test sets.

☁ Distributed Systems and Cloud Computing

The system is containerized with Docker and orchestrated via Docker Compose to spin up ZooKeeper and multiple clients and run controlled crash tests.

💻 Computer Architecture and Programming Systems

Implementation of the Fish School Search (FSS) metaheuristic (with individual, instinctive, volitive phases) for optimization, with the numeric core written in x86 Assembly and C. Two execution paths were built: 32-bit with SSE and 64-bit with AVX, applying SIMD vectorization (e.g., SUBS, MULPS, HADDPS, SHUF), loop unrolling, and cache-friendly row-major memory access. OpenMP provides multi-threaded parallelism. Critical kernels were ported to Assembly and integrated with the C FSS functions.

  • 🛠️ Computer Architecture Concepts
  • 🛠️ SIMD Vectorization
  • 🛠️ x86 Assembly 32-bit/64-bit
  • 🛠️ Low-Level Optimization
  • 🛠️ OpenMP
  • 🛠️ C programming
  • 🛠️ Low-level integration
  • 🛠️ Interaction: Hardware-level code and Higher-level C routines
