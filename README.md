## 🧭 Overview

This repository contains the source code of my personal portfolio and CV website.

The site presents my professional identity as an AI/ML Engineer: focus on NLP, RAG and Document AI pipelines, research in Machine Unlearning and Responsible AI, and a background that combines technical depth with leadership and community experience.

The visual identity takes inspiration from the Windows BSOD — a deliberately playful tech reference that reflects the tone of the site: precise, a little self-aware, and not afraid of personality.

## ✨ Features

- **BSOD-inspired hero** with parodic structure faithful to the Windows error screen.
- **Single-page responsive layout** — mobile, tablet and desktop.
- **Fixed floating navbar** with glass-morphism effect, hamburger-to-X animation and smooth scroll to section headings with precise offset.
- **Scroll animations** via AOS, hero animated on load via CSS keyframes (no scroll dependency).
- **Project cards** with curated hard skill chips and a "+N altre" expand/collapse for long lists.
- **Research card** with journal citation, author list and DOI link.
- **Tech Stack** organized by category, tools only (no concepts or datasets).
- **Reduced-motion support** for users who prefer fewer animations.
- **SEO and social sharing** — meta description, Open Graph, Twitter cards and JSON-LD structured data.
- **GitHub Pages ready** with no build step.

## 🛠️ Site Tech Stack

The site is a lightweight static build — no framework, no bundler, no package manager.

| Layer | Technology |
|---|---|
| Structure | HTML5 (semantic) |
| Styling | CSS3 with custom design tokens, responsive layout |
| Behaviour | Vanilla JavaScript (ES6+) |
| Grid/utilities | Bootstrap 4.4.1 (CSS + JS, minified, bundled locally) |
| Animations | AOS 2.3.1 (CDN) + CSS keyframes for above-fold elements |
| Heading font | Montserrat via Google Fonts |
| Body font | Selawik Light (self-hosted, SIL OFL 1.1) |
